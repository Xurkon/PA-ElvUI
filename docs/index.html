✨ Features

🎯 Action Bars

Fully customizable action bars with ButtonFacade skin support for 30+ button styles.

👤 Unit Frames

Player, target, party, and raid frames with extensive customization options.

🏷️ Nameplates

Enhanced nameplates with styling, filtering, and threat indicators.

🎒 Bags

Unified bag interface with sorting, searching, and item categorization.

💬 Chat

Enhanced chat with URL detection, copy/paste, and customization.

🗺️ Maps

Customizable minimap and world map with coordinates and zoom.
